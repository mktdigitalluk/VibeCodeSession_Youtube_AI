# 🚀 VibeCode Sessions AI — Automated YouTube Content Engine

An end-to-end AI-powered pipeline that generates, produces, and publishes long-form music videos and short-form content (Reels/Shorts) for a faceless YouTube channel — fully automated.

---

## 🎯 Project Overview

This project was designed to solve a real-world challenge:

> How to scale content production for monetizable YouTube channels using AI — with minimal manual intervention.

The system autonomously:
- Generates video ideas using LLMs
- Creates AI music tracks
- Produces visual assets (thumbnails / scenes)
- Renders long-form videos (1h–3h)
- Generates short-form reels for growth
- Publishes content to YouTube via API

---

## 🧠 Core Concept

A **faceless YouTube channel for developers**, combining:
- Focus music 🎧
- Immersive environments 🌄
- AI-generated content 🤖

---

## 🏗️ Architecture

LLM (Gemini) → Idea Agent → Image + Music Services → Video Rendering (FFmpeg) → YouTube Upload

---

## ⚙️ Tech Stack

- Python 3.12
- Google Gemini API
- Replicate (Flux image generation)
- Kie.ai (music generation)
- FFmpeg
- YouTube Data API

---

## 🔁 Automation Flow

1. Generate idea  
2. Generate thumbnail  
3. Generate music  
4. Loop audio  
5. Render video  
6. Upload to YouTube  
7. Generate reels  
8. Save history
